Detailed Description

template<typename T>
class meevax::type_traits::is_scoped_bitmasks< T >

テンプレートパラメータの型がビット幅演算をサポートしているか否かを返すメタ関数。

デフォルトでは常に std::false_type から派生する。
つまり、ビット演算をサポートするスコープを持つ列挙型を作成した場合、 そのクラスに対するこのメタ関数の特殊化を定義しておかなければならない。

なお、このメタ関数が true を返す列挙型に関して、 各メンバの値が 2^n の要件を満たしているか否かはチェックされない。
